# houndex-evals (Python)

Deterministic regression/evaluation harness built on `houndex-core`: fixture
schema, a domain-agnostic `OutputEnvelope` rubric (trace resolution, envelope
validity, canonical-JSON determinism), and markdown report generation. Mirrors
the TypeScript `houndex/evals` package.

See the [repository README](../../README.md) for the project overview.
